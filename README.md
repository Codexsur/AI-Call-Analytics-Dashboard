# AI Call Analytics Dashboard

A modern React.js-based dashboard for monitoring, searching, filtering, and analyzing customer call logs and transcripts. The application provides an intuitive interface for managing call records, viewing transcript data, and efficiently navigating large datasets through advanced search and filtering capabilities.

---

## Features

* Real-Time Transcript Search
* Customer Call Log Monitoring
* Date Range Filtering
* Pagination for Large Datasets
* Responsive User Interface
* Audio Recording Tracking
* REST API Integration
* Expandable Transcript Viewer
* Call Direction Classification
* Label-Based Categorization
* Optimized Data Presentation

## Project Overview

AI Call Analytics Dashboard is a React.js-based application designed to help organizations monitor, search, and analyze customer call records efficiently. The platform provides transcript-based search, date-range filtering, pagination, and API-driven data visualization for handling large-scale call datasets.

Users can:

* Search customer call transcripts instantly
* Filter call records using date ranges
* Monitor incoming and outgoing calls
* View transcript details and audio references
* Analyze customer interaction logs
* Navigate large datasets efficiently
* Track call history through a structured dashboard

## Installation

### Prerequisites

* Node.js (v14 or higher)
* npm

### Clone Repository

```bash
git clone https://github.com/yourusername/AI-Call-Analytics-Dashboard.git
cd AI-Call-Analytics-Dashboard
```

### Install Dependencies

```bash
npm install
```

### Run Application

```bash
npm start
```

The application will be available at:

```bash
http://localhost:3000
```
